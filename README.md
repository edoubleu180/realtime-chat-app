# Realtime Chat App (Node.js + Socket.IO)

A simple but professional realtime chat application built with Node.js, Express, Socket.IO, and vanilla JavaScript.

## Features

- Realtime messaging with WebSockets
- Join/leave notifications
- Online users list
- Basic username system
- Environment-based config (PORT, client origin)

## Tech Stack

- Node.js, Express
- Socket.IO
- Vanilla JS, HTML, CSS

## Running the project

### 1. Server

```bash
cd server
cp .env.example .env   # edit if needed
npm install
npm run dev            # or: npm start
